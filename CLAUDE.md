# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

**Montportfolio** - A static portfolio website built with HTML, CSS, and vanilla JavaScript. No frameworks.

## Design Guidelines

- Minimalist and professional aesthetic
- Dark background theme
- Mobile-first responsive design

## Code Conventions

- **Code language**: English (variables, functions, comments, file names)
- **Content language**: French (all visible text, labels, titles, descriptions)

Example:
```javascript
// Good
const projectList = document.querySelector('.projects');
const showMoreButton = document.getElementById('show-more');

// Content in HTML
<h1>Bienvenue sur mon portfolio</h1>
<button id="show-more">Voir plus</button>
```
